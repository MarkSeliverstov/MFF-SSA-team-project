# NSWI130 Schedules System (SCH)

## Main info about system

- **stakeholders:**
  - students
  - teachers
  - schedulin committee

- **feature areas:**
  - scheduling (by committee and teachers)
  - management of schedule preferences (by teachers)
  - viewing schedules
  - statistical reports (occupancy of rooms,...)
